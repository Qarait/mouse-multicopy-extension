# Privacy Policy

Effective date: May 29, 2026

Mouse MultiCopy is designed to keep copied text local to your browser.

## Data The Extension Handles

Mouse MultiCopy can store text that you highlight on webpages while collect mode is enabled. It may also store the page title, page URL, and capture time for each saved slot.

## How Data Is Stored

Saved snippets are stored using browser extension local storage on your device.

## Data Sharing

Mouse MultiCopy does not send captured text, URLs, page titles, or usage data to any server. Mouse MultiCopy does not sell, rent, or share user data.

## Permissions

Mouse MultiCopy requests access to webpages so it can read your current text selection and show the floating slot palette. It requests clipboard write access so it can copy a selected slot when no text field is focused.

## User Control

You can pause collect mode, clear saved slots, or uninstall the extension at any time. Clearing saved slots removes the snippets stored by the extension.
