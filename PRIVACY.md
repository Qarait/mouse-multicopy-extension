# Privacy Policy

Effective date: June 12, 2026

Mouse MultiCopy is designed to keep copied and highlighted text local to your device.

## Data Mouse MultiCopy Handles

The browser extension can store text that you intentionally highlight on webpages while collection is enabled. The Windows desktop app can access and store text that you intentionally copy to the Windows clipboard while collection is enabled.

Copied or highlighted text may contain personal information chosen by the user. Mouse MultiCopy may also store highlight names, order, page numbers entered by the user, local session names, output preferences, page titles, page URLs, and capture times.

## How Data Is Stored

Saved snippets and settings are stored locally on the user's device using browser extension local storage or the Windows application data folder, depending on the product version.

## Data Sharing

Mouse MultiCopy does not send captured text, URLs, page titles, or usage data to the developer or an external server. Mouse MultiCopy does not sell, rent, or share user data with third parties.

## Permissions

Mouse MultiCopy runs on webpages so it can read the text you intentionally highlight, show the floating slot palette, and paste a selected slot into focused text fields. It requests clipboard write access so it can copy a selected slot when no text field is focused.

The Windows app monitors clipboard text only while collection is enabled so it can create numbered highlights from text the user copies.

## User Control

You can pause collection, delete or clear saved highlights, or uninstall Mouse MultiCopy at any time. Clearing saved highlights removes the snippets stored by Mouse MultiCopy.
