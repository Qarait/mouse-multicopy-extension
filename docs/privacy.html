Privacy Policy

Mouse MultiCopy is designed to keep copied and highlighted text local to your device.

Data Mouse MultiCopy handles

The browser extension can store text that you intentionally highlight on webpages while collection is enabled. The Windows desktop app can access and store text that you intentionally copy to the Windows clipboard while collection is enabled. This text may contain personal information chosen by the user.

Mouse MultiCopy may also store highlight names, order, page numbers, local session names, output preferences, page titles, page URLs, and capture times.

Storage and sharing

Saved snippets and settings are stored locally on the user's device. Mouse MultiCopy does not send captured text, URLs, page titles, or usage data to the developer or an external server. It does not sell, rent, or share user data with third parties.

Permissions and user control

Mouse MultiCopy accesses selected webpage text or Windows clipboard text only to provide its highlight collection and paste features. You can pause collection, delete or clear saved highlights, or uninstall Mouse MultiCopy at any time.
